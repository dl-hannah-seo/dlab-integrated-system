# 홈 대시보드 설계

**날짜**: 2026-06-16  
**상태**: 승인됨  
**라우트**: `/dashboard`

---

## 개요

로그인 후 첫 번째로 보이는 홈 화면. 원장이 하루를 시작할 때 오늘 출결 현황, 수업 일정, 납부 상태를 한눈에 파악하고 자주 쓰는 기능으로 빠르게 이동할 수 있도록 한다.

기존 손익 대시보드(`/dashboard`)는 별도 라우트로 이전되었으므로 이 페이지가 어드민 홈을 대체한다.

## 레이아웃 구조

### ① 다크 인사 배너 (풀 width)

배경색 `#37352F`. 좌우 2구역으로 나뉨.

**왼쪽**
- 캠퍼스명 · 오늘 날짜 (보조 텍스트, `#9B9B97`)
- 인사 문구: "좋은 아침이에요, 원장님 👋" (흰색 굵게)
- 오늘 수업 수 · 재원생 수 (보조 텍스트)

**오른쪽**
- 오늘 출석률 (%) — 큰 숫자, `#0F7B6C` 강조색
- 출석 인원 / 전체 인원 (보조 텍스트)

### ② 2단 본문 (grid: 1fr / 1.6fr)

**왼쪽 컬럼**

1. **오늘 수업 카드**
   - 오늘 요일에 해당하는 수업 목록
   - 각 행: 시간 · 반명 · 수강 인원
   - 수업 없으면 "오늘 수업이 없습니다" 빈 상태 표시

2. **빠른 실행 구역**
   - 오렌지 배경 카드(`#FFF1EC`)로 공간 확보
   - 현재는 비워둠 — 추후 액션 버튼 추가 예정

**오른쪽 컬럼**

1. **주간 출결 추이 카드**
   - 이번 주 월~일 바 차트 (SVG 없이 CSS flex로 구현)
   - 오늘 이후(미래) 요일은 `#E9E9E7` 흐린 색으로 표시
   - 각 바 하단에 요일 레이블

2. **납부 현황 + 미납 원생 카드** (2열 그리드)
   - 왼쪽: 도넛 차트 (완납/미납/부분납 비율), 중앙에 완납률 %
     - 완납: `#FF6C37`, 미납: `#EB5757`, 부분납: `#E9E9E7`
   - 오른쪽: 미납 원생 목록 (이름 · 반 일정), 문자 발송 버튼

## 데이터 소스 (mock-data.ts)

| UI 구역 | 사용 데이터 |
|---------|------------|
| 인사 배너 출석률 | `dashboardData.today_attend / total_students` |
| 오늘 수업 목록 | `classes` → `schedule` 필드에 오늘 요일 약자 포함 여부로 필터 (`"토 09:00"` → `"토"` 포함 여부, `"화목 16:00"` → `"화"` 또는 `"목"` 포함 여부) |
| 주간 출결 추이 | `attendance` 레코드를 이번 주 월~일 날짜별 집계 |
| 납부 도넛 차트 | `dashboardData.paid_students / unpaid_students / total_students` |
| 미납 원생 목록 | `getUnpaidStudents()` |

## 컴포넌트 구성

```
app/(admin)/dashboard/page.tsx   ← 페이지 (Client Component)
  ├── GreetingBanner             ← 인라인 또는 별도 컴포넌트
  ├── TodayClassList             ← 왼쪽 상단
  ├── QuickActionsZone           ← 왼쪽 하단 (빈 상태)
  ├── WeeklyAttendanceChart      ← 오른쪽 상단
  └── PaymentStatusRow           ← 오른쪽 하단 (도넛 + 미납 목록)
```

기존 `Card`, `Button`, `Badge` 컴포넌트 재사용. 차트는 외부 라이브러리 없이 CSS flex/conic-gradient로 구현.
