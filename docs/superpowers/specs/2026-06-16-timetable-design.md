# 시간표 페이지 디자인 스펙

**날짜:** 2026-06-16  
**상태:** 승인됨  
**라우트:** `/schedule` (추후 `/timetable`로 이전 예정)

---

## 배경 및 목적

현재 사이드바의 `/timetable`은 **"출석 체크"** 기능을 담당하며, 실시간 출결 관리가 주목적이다. 이와 별개로 학원 운영자가 **학기 전체 수업 편성을 한눈에 조회**할 수 있는 시간표 페이지가 필요하다.

출석 체크 메뉴는 장기적으로 제거될 예정이므로, 신규 시간표 페이지를 `/schedule`로 우선 추가하고 이후 `/timetable`로 라우트를 이전한다.

---

## 결정 사항

| 항목 | 결정 | 근거 |
|------|------|------|
| 진입점 | 사이드바에 "시간표" 항목 독립 추가 | 출석 체크와 역할 분리 |
| 라우트 | `/schedule` → 추후 `/timetable` 이전 | 기존 출석 체크 라우트 충돌 회피 |
| 기본 뷰 | **주간 그리드** (요일 × 시간대 격자) | 정기 편성 파악에 최적 |
| 보조 뷰 | **요일별 카드 리스트** (토글 전환) | 수업 수 증가 시 가독성 확보 |
| 블록 표시 정보 | 수업명 · 담당 선생님 · 현재 수강 인원 | 운영에 필요한 핵심 3가지 |
| 클릭 동작 | **팝오버**로 반 상세 요약 표시 | 페이지 이탈 없이 빠른 확인 |
| 색상 체계 | 토요일 오렌지(`#FF6C37`), 화목 블루(`#3B82F6`) | 기존 출석 체크 색상 체계 유지 |

## 페이지 구조

### 헤더
- 제목: "시간표"
- 부제: "[년도] [학기] · 판교 캠퍼스"
- 우측: **학기 선택 드롭다운** + **그리드/카드 토글**

### 주간 그리드 뷰 (기본)

```
        월    화    수    목    금    토(오렌지)
09:00  [ ]  [ ]  [ ]  [ ]  [ ]  [파이썬기초 / 론 / 14명]
10:00  [ ]  [ ]  [ ]  [ ]  [ ]  [파이썬기초 / 민 / 15명]
11:00  [ ]  [ ]  [ ]  [ ]  [ ]  [맞춤수업   / 론 / 11명]
16:00  [ ]  [파이썬/김/18]  [ ]  [파이썬/김/18]  [ ]  [ ]
17:00  [ ]  [맞춤/박/16]   [ ]  [맞춤/박/16]   [ ]  [ ]
18:00  [ ]  [아두이노/이/4] [ ]  [아두이노/이/4] [ ]  [ ]
```

- 수업이 없는 셀: 빈 상태 (배경 흰색)
- 수업 블록: 담당 선생님 요일 색상 기반 배경
- 같은 반이 화·목 등 복수 요일에 걸칠 경우 각 셀에 동일 블록 반복 표시

### 요일별 카드 뷰 (토글 시)

요일 그룹 카드로 나열:
```
[토요일]
  09:00  파이썬기초 · 론 선생님 · 14명
  10:00  파이썬기초 · 민 선생님 · 15명
  11:00  맞춤수업   · 론 선생님 · 11명

[화요일 · 목요일]
  16:00  파이썬기초 · 김 선생님 · 18명
  17:00  맞춤수업   · 박 선생님 · 16명
  18:00  아두이노   · 이 선생님 ·  4명
```

### 팝오버 (블록 클릭 시)

수업 블록 클릭 시 팝오버 표시 (모달 아님, 페이지 이탈 없음):

```
┌─────────────────────────┐
│ 파이썬기초          ✕  │
│ 📅 토요일 09:00         │
│ 👨‍🏫 론 선생님           │
│ 👥 14명 / 정원 20명     │
│ 📆 2026.04.01 ~ 07.31   │
│ 💰 월 120,000원         │
├─────────────────────────┤
│ 반 상세 페이지로 이동 → │
└─────────────────────────┘
```

- "반 상세 페이지로 이동" 클릭 시 `/classes/[id]`로 이동

## 데이터 소스

기존 `lib/mock-data.ts`의 타입 및 데이터를 그대로 활용:

- `ClassGroup` — 요일 그룹(`day_group`) · 시간대(`time_slot`)
- `Class` — 수업명(`course`) · 담당(`teacher`) · 정원(`capacity`) · 기간(`start_date`, `end_date`) · 수강료(`tuition_fee`)
- `Semester` — 학기 선택 드롭다운 목록
- 수강 인원은 `students` 배열에서 `class_id` 기준으로 집계

출석 체크 메뉴 제거 시:
1. `/timetable` 라우트를 시간표 페이지로 교체
2. 사이드바 항목의 `href`를 `/timetable`로 변경
3. `/schedule` 라우트 제거 또는 리다이렉트
