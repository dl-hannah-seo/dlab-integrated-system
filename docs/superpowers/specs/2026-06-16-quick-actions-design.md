**날짜:** 2026-06-16

---

## 배경 및 목표

출석체크와 문자발송은 데스크 담당자가 하루에도 여러 번 사용하는 빈도 높은 기능이다. 현재 출석체크는 `/timetable` 전용 페이지로 구현되어 있으나, 이는 페이지 이동을 요구하고 UX 흐름을 끊는다. 문자발송 기능은 아직 없다.

**목표:** 두 기능을 슬라이드-오버 패널로 전환하여 어느 페이지에서도 컨텍스트 이탈 없이 사용 가능하게 한다.

---

## 아키텍처 결정

### 제거: `/timetable` 페이지 (출석체크 전용 페이지)
- 키오스크 자동 출석이 주 흐름이므로 전용 페이지는 과함
- 슬라이드-오버 패널로 대체
- `/attendance` (출결 현황) 페이지는 통계/이력 확인용으로 유지

### 추가: 사이드바 "빠른 실행" 섹션
- 기존 nav 메뉴와 유저 정보 사이에 구분선으로 구역 추가
- 아이콘 + 레이블 형태의 2개 버튼

### 추가: 출석체크 슬라이드-오버 패널
### 추가: 문자발송 슬라이드-오버 패널

## 컴포넌트 설계

### 1. 사이드바 변경 (`components/layout/Sidebar.tsx`)

```
[기존 nav 메뉴]
─ 키오스크 화면 링크
────────────────── (구분선)
⚡ 빠른 실행          ← 섹션 레이블
[✓] 출석 체크         ← 패널 트리거
[✉] 문자 발송         ← 패널 트리거
────────────────── (구분선)
[유저 정보 / 로그아웃]
```

- 스타일: 일반 nav와 구분되도록 옅은 오렌지 tint (`#FFF1EC`) 배경
- `/timetable` 항목은 nav에서 제거

### 2. 출석체크 패널

**트리거:** 사이드바 "출석 체크" 버튼

**동작 원리:**
- 키오스크에서 학생이 로그인하면 자동 출석 처리됨
- 패널은 실시간 현황 조회 + 미도착 학생 수동 처리 역할

**레이아웃:**
```
┌─────────────────────────┐
│  출석 체크              ✕ │
│  ─────────────────────── │
│  반 선택  [월 19:00 코딩 ▼]│
│  수업시작 19:00 · 현재 19:14│
│                          │
│  ✅ 출석 (키오스크)  4명  │
│  김민준 · 이서연 · 박지호 · 최수아│
│                          │
│  ⚠️ 미도착              2명 │
│  홍길동   [지각으로처리][결석]│
│  김영희   [지각으로처리][결석]│
│                          │
│  ─────────────────────── │
│       [미도착 2명 문자발송]│
└─────────────────────────┘
```

**저장 방식: 즉시 저장 (per-student)**
- 지각/결석 버튼 클릭 즉시 서버에 저장 (낙관적 업데이트)
- 별도 "저장" 버튼 없음 → 실수로 패널 닫아도 데이터 유실 없음
- 저장 후 출결 현황 페이지에 실시간 반영

**권한:**
| 액션 | 데스크 스태프 | 담임 | 팀장 |
|------|:-----------:|:----:|:----:|
| 패널 열기·현황 보기 | ✓ | ✓ | ✓ |
| 미도착 → 지각/결석 처리 | ✓ | ✓ | ✓ |
| 이미 저장된 상태 수정 | ✗ | ✓ | ✓ |

- 담임/팀장 역할은 기존 시스템의 역할 구분 활용

**출결 현황 페이지 수정 기능:**
- 현재 미구현 상태. 담임/팀장 권한의 수정 기능은 향후 별도 추가 예정.

### 3. 문자발송 패널

**트리거 3가지 및 자동 세팅:**

| 트리거 위치 | 수신자 자동 세팅 | 템플릿 자동 선택 |
|------------|----------------|----------------|
| 출석 패널 "미도착 N명 문자발송" | 결석·지각 학생 부모 | 결석 알림 |
| 수납 페이지 "미납 안내 문자" | 미납 학생 부모 | 미납 안내 |
| 사이드바 "문자 발송" 직접 | 빈 상태 (직접 선택) | 직접 입력 |

**레이아웃:**
```
┌─────────────────────────┐
│  문자 발송              ✕ │
│  ─────────────────────── │
│  수신자                  │
│  ┌─────────────────────┐ │
│  │ 홍길동 부모님  ✕     │ │
│  │ 김영희 부모님  ✕     │ │
│  │ + 수신자 추가        │ │
│  └─────────────────────┘ │
│                          │
│  템플릿                   │
│  ○ 결석 알림             │
│  ○ 미납 안내             │
│  ● 직접 입력             │
│                          │
│  ┌─────────────────────┐ │
│  │ 메시지 내용...       │ │
│  └─────────────────────┘ │
│  0 / 90자               │
│                          │
│           [발송하기]      │
└─────────────────────────┘
```

- 수신자: 태그형 입력 (개별 삭제 가능, 추가 가능)
- 템플릿 선택 시 메시지 자동 채워짐 (수동 수정 가능)
- 90자 카운터 (표준 SMS 1건 기준)
